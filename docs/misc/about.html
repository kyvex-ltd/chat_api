Powering Kyvex Chat

What is Chat API?

The API serving Kyvex Chat: an open source alternative to Discord. This API accounts for all the backend functionality of Kyvex Chat

Chat API is currently in beta, and is subject to change.

Powered by:

  • Express
  • JSON Web Tokens
  • MongoDB
  • Mongoose
